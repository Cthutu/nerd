# Image Viewer Demo Assets

These assets are generated placeholder images used by the image viewer example
until `std.image` can load richer fixtures.

Downloaded with:

```sh
wget -O examples/image_viewer/assets/demo.png 'https://placehold.co/320x200/png?text=Nerd+PNG'
wget -O examples/image_viewer/assets/demo.jpg 'https://placehold.co/320x200.jpg?text=Nerd+JPEG'
```

Files:

- `demo.png`: 320x200 PNG, RGBA.
- `demo.jpg`: 320x200 baseline JPEG, RGB.

Source:

- <https://placehold.co/>
